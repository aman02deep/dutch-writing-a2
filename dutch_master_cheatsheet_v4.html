3 rules — memorise only these
Rule 1 — omdat → verb to END
❌ omdat ik kan niet komen
✅ omdat ik niet kan komen
✅ omdat ik ziek ben / thuis moet blijven
Rule 2 — mijn = my / mij = me
✅ Mijn naam / Mijn telefoonnummer (possession)
✅ Kunt u mij bellen? (object)
Rule 3 — formal vs informal
✅ Kunt u mij bellen? — formal
✅ Kun je me bellen? — informal
❌ Never mix in same email!

Questions — t-drop
SubjectRuleExample
je / jijt drops!Werk je? (not werkt jij!)
ut staysWerkt u?
hij / zijt staysWerkt hij?
Perfect tense — GKBV
GKBV → zijn: Gaan Komen Blijven Vertrekken
✅ Ik ben gegaan / gekomen / gebleven / vertrokken

Everything else → hebben:
✅ Ik heb een wasmachine gekocht. Ik heb veel gewerkt.

Past form always at END!
gaan → gegaan
komen → gekomen
blijven → gebleven
vertrekken → vertrokken
kopen → gekocht
werken → gewerkt
